Project Details:

In this assignment we will be working with Angular, Mongoose and Express Session. Using the Angular $http service, we are able to make AJAX requests to our server-side routes, whom pass data to our server side controller, which does the actual database querying.

This data is then sent back as an http response, in the form of JSON data, which can then be utilized in the angular factory to update our angular controller and Views. By using $http promises, we're able to make requests to the database and update the DOM upon a response from the server, real-time without a page reload (yay AJAX).
